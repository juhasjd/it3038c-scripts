Date output Powershell script
=============================

Resources needed
-----------------
windows powershell script application (download with https://github.com/PowerShell/PowerShell/releases/download/v7.3.2/PowerShell-7.3.2-win-x64.msi)
the file datescript.ps1




To run script type the following in the powershell command line
.\datescript.ps1
